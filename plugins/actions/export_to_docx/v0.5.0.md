# Export to Word Enhanced — v0.5.0 Release Notes

**Release Date:** 2026-05-03
**Author:** [Fu-Jie](https://github.com/Fu-Jie)
**Version:** 0.5.0

---

## What's New

### 🔠 Heading-Level Font Customization

New Valves let you configure H1 and H2 fonts independently from body text:

| Valve | Default | Description |
| :--- | :--- | :--- |
| `FONT_H1_LATIN` | `Aptos Display` | Latin font for Heading 1 |
| `FONT_H1_ASIAN` | `SimHei` | Asian font for Heading 1 |
| `FONT_H2_LATIN` | `Calibri Light` | Latin font for Heading 2 |
| `FONT_H2_ASIAN` | `SimHei` | Asian font for Heading 2 |

Empty values fall back to `FONT_LATIN` / `FONT_ASIAN`. Both `export_to_word.py` (English Valves) and `export_to_word_cn.py` (Chinese Valves) are updated in sync.

### ⬆️ Relative Heading Algorithm

When the exported content has no `# H1`, the plugin now scans for the minimum heading level and promotes all headings by the appropriate offset. For example, if the content starts at `##`, all `##` become `#` and `###` become `##`. A debug log is emitted when promotion occurs.

### 🧹 Metadata Sanitization

The generated Word document now sets `author`, `last_modified_by`, `title`, and other core properties to the real user name and chat title, removing the default `python-docx` library fingerprints that previously leaked into document properties.

### 🎨 Heading Style Fixes (Critical)

- **Theme font clearing**: Cleared `w:asciiTheme`, `w:hAnsiTheme`, `w:eastAsiaTheme`, and `w:cstheme` on default style, Title style, and Heading 1-9 styles. This prevents Word from overriding explicit font choices with theme defaults (e.g., SimSun → MS Gothic).
- **Run-level CJK enforcement**: For Heading paragraphs, the East Asian font is now set at the Run level (`w:eastAsia` + cleared `w:eastAsiaTheme`), resolving cases where style-level fonts alone were unreliable.
- **Removed spurious underlines and borders**: Heading styles no longer inherit underline or paragraph border decorations from the default template.

---

## Migration Notes

- **No manual migration required.** Existing users can simply update the plugin.
- New Valves have sensible defaults (`Aptos Display` / `Calibri Light` / `SimHei`) — no configuration needed unless you want to customize.
- If you had previously set custom H1/H2 fonts via CSS hacks, you can now use the native Valves instead.
