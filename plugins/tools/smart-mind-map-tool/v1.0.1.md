# Smart Mind Map Tool v1.0.1 Release Notes

## Overview

Smart Mind Map Tool v1.0.1 is a maintenance release focused on OpenWebUI 0.9.x compatibility. It keeps the existing proactive mind map experience unchanged while making database access work across both synchronous and asynchronous backend runtimes.

## Highlights

- Version-aware database access now adapts to the active OpenWebUI runtime.
- User lookup calls now work safely on OpenWebUI 0.9.x without breaking older deployments.
- Mind map generation, i18n UI, toolbar controls, and export behavior remain unchanged.

## Upgrade Notes

- No migration steps are required.
- No new Valves are introduced.
